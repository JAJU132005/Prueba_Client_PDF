variante · Imágenes a PDF (/images-to-pdf) — misma anatomía

Álbum de recortes ¡ZAS!

  • Dropzone acepta imágenes; cada una aparece como foto pegada con cinta en el cuaderno.
  • Reordenar = despegar y re-pegar la foto (mismas flechas del FileList).
  • Botón de acción: «Cerrar el álbum con moño».
  • Imagen corrupta → foto con panda confundido + mensaje real de error en bocadillo (sin reescribirlo).

notas para el implementer

Compartidos que embebe: Dropzone(multi), FileList (+PageCountBadge «N páginas»), PreviewModal, ProgressBar, ResultPanel. La acción llama a `merge` / `imagesToPdf` existentes; el bambú consume el progreso del worker tal cual.
